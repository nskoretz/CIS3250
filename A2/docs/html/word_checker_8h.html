Function to recursively check every adjacent piece on a board and see if it is the next character needed by the word being sought after. If it is, move onto the next piece to be found and recursively call this function until all characters are found (or not found)

Precondition
gameBoard and visited must exist, i, j, and subLen must be non-negative real numbers,
Returns
integer whether the word was found or not on the board
Parameters
iinteger representing the row in the game board
jinteger representing the coloumn in the game board
wordstring containing the word being sought after on the board
gameBoardpointer to a gameBoard array storing the character and position of each index in the board (2D array)
subLeninteger representing the last index from a string whose character has been found on the game board
visitedpointer to an array of characters that have already been found
